Getting Started
These instructions will help you get a copy of the project up and running on your local machine for development and testing purposes.

Prerequisites
* Node.js v14+ (recommended v16+)
* npm or yarn

  Installation

  # clone the repo
git clone https://github.com/bhupendrar1/Health-Monitoring-App.git
cd health-monitoring-app


# install dependencies (npm)
npm install
# OR (yarn)
yarn install

Running Locally
Open http://localhost:3000 in your browser (or the port shown in terminal).
